# RadioKit Widgets

A Flutter widget library providing hardware-control UI widgets — buttons, sliders, knobs, joysticks, LEDs, switches, and more — designed for use with the [RadioKit](https://github.com/rambros3d/RadioKit) Arduino library.

## Widgets

| Widget | Description |
|---|---|
| `RKButton` | Momentary or toggle push button |
| `RKSwitch` | On/off toggle switch |
| `RKSlider` | Linear slider (horizontal/vertical) |
| `RKKnob` | Rotary knob |
| `RKJoystick` | 2-axis joystick |
| `RKLed` | LED indicator |
| `RKMultiButton` | Radio-style multi-button group |
| `RKMultiSelect` | Bitmask multi-select group |
| `RKDisplay` | Text display output |

## Getting Started

```yaml
dependencies:
  radiokit_widgets: ^0.0.1
```

```dart
import 'package:radiokit_widgets/radiokit_widgets.dart';
```
